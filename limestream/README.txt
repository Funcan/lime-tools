Introduction:

Welcome to the LimeStream software.  LimeStream is a simple receiver/transmiter IQ stream.

Usage: 
  -f <Frequency> : Frequency in Hz
  -s <Sample rate> : Sample Rate in Hz 
  -g <Gain (normalized)> : Normalize gain (0.0 to 1.0)
  -r <filename> : Receive stream into IQ s16 file (use '-' for stdout)
  -t <filename> : Transmit stream from IQ s16 file (use '-' for stdin).
  -b <Buffer latency> maximum Buffer latency in milliseconds
  -d <Device index> set the Device index if multiple devices
  -p <calibration file> : use a Profile calibration file 
  -c <calibration file> : generate a profile Calibration file 
  -v : Verbose , print info and statistics 

Compatible with LimeSDR and LimeSDRmini.
